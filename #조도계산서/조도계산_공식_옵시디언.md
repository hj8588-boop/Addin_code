# 조명 조도 계산식

## 1. 평균조도

실 전체의 평균 조도를 구할 때 사용한다.

$$
E = \frac{F \times N \times U \times M}{A}
$$

| 기호 | 의미 | 단위 |
|---|---|---|
| $E$ | 평균조도 | lx |
| $F$ | 등기구 1대 광속 | lm |
| $N$ | 등기구 수량 | 대 |
| $U$ | 조명률 | - |
| $M$ | 보수율 | - |
| $A$ | 실 면적 | m² |

```text
평균조도 = 등기구광속 × 등기구수 × 조명률 × 보수율 / 면적
```

## 2. 필요 등기구 수

목표 조도를 만족하기 위해 필요한 등기구 수를 계산할 때 사용한다.

$$
N = \frac{E \times A}{F \times U \times M}
$$

실제 등기구 수는 소수점으로 설치할 수 없으므로 올림 처리한다.

$$
N_{required} = \lceil \frac{E \times A}{F \times U \times M} \rceil
$$

```text
필요등수 = CEILING((요구조도 × 면적) / (광속 × 조명률 × 보수율))
```

현재 Dynamo/Revit 계산 스크립트의 핵심 공식도 이 식이다.

```python
raw_count = (required_lux * area) / (fixture_flux * utilization * maintenance)
required_count = ceil(raw_count)
```

## 3. 계산조도

올림 처리된 실제 등기구 수로 다시 계산한 조도이다.

$$
E_{calculated} = \frac{F \times U \times M \times N_{required}}{A}
$$

```text
계산조도 = 광속 × 조명률 × 보수율 × 필요등수 / 면적
```

이 값은 목표 조도보다 같거나 높게 나오는 것이 일반적이다.

## 4. 실지수

실의 형상과 광원 높이를 반영하는 값이다.

$$
K = \frac{A}{H_m \times (L + W)}
$$

| 기호 | 의미 | 단위 |
|---|---|---|
| $K$ | 실지수 | - |
| $A$ | 실 면적 | m² |
| $H_m$ | 작업면 기준 광원고 | m |
| $L$ | 실 길이 | m |
| $W$ | 실 폭 | m |

```text
실지수 = 면적 / (광원고 × (가로 + 세로))
```

## 5. 조명률 적용 기준

현재 스크립트는 실지수에 따라 조명률을 아래처럼 자동 산정한다.

| 실지수 범위 | 조명률 |
|---:|---:|
| $K \le 0.75$ | 0.60 |
| $0.75 < K \le 1.00$ | 0.68 |
| $1.00 < K \le 1.25$ | 0.76 |
| $1.25 < K \le 1.50$ | 0.81 |
| $1.50 < K \le 2.00$ | 0.88 |
| $2.00 < K \le 2.50$ | 0.93 |
| $2.50 < K \le 3.00$ | 0.96 |
| $K > 3.00$ | 1.01 |

## 6. 점조도

특정 한 점에서의 조도를 간단히 계산할 때 사용하는 식이다.

$$
E = \frac{I}{d^2}
$$

입사각을 고려하면 다음과 같다.

$$
E = \frac{I \times \cos \theta}{d^2}
$$

| 기호 | 의미 | 단위 |
|---|---|---|
| $E$ | 조도 | lx |
| $I$ | 광도 | cd |
| $d$ | 광원과 계산점 사이 거리 | m |
| $\theta$ | 입사각 | degree |

## 7. 예시

조건:

| 항목 | 값 |
|---|---:|
| 요구조도 | 500 lx |
| 면적 | 100 m² |
| 등기구 광속 | 4,000 lm |
| 조명률 | 0.60 |
| 보수율 | 0.80 |

계산:

$$
N = \frac{500 \times 100}{4000 \times 0.60 \times 0.80}
$$

$$
N = 26.04
$$

따라서 필요 등기구 수는 다음과 같다.

```text
필요등수 = 27대
```

27대를 적용했을 때 계산조도:

$$
E_{calculated} = \frac{4000 \times 0.60 \times 0.80 \times 27}{100}
$$

$$
E_{calculated} = 518.4lx
$$

## 8. Revit 공유 파라미터 연결

현재 조도 계산 흐름에서 주로 쓰는 파라미터는 아래와 같다.

| 용도 | 파라미터 |
|---|---|
| 목표 조도 | `필수 요구조도` |
| 조명률 | `조명율` |
| 보수율 | `보수율` |
| 적용 전등 타입 | `조도_적용전등타입` |
| 직접 입력 광속 | `조도_광속_lm` |
| 계산 결과 등수 | `필요등수` |
| 적용 광속 기록 | `fixtureFlux_lm` |
| 적용 기구 기록 | `조도_적용기구` |
| 실지수 기록 | `조도_실지수` |

광속은 우선순위에 따라 읽는다.

```text
Space.조도_광속_lm
→ Space.fixtureFlux_lm
→ Lighting Fixture의 fixtureFlux_lm 또는 광속 파라미터
```

## 9. 반사율

현재 계산 스크립트에서는 반사율을 결과 기록용 고정값으로 사용한다.

| 항목 | 값 |
|---|---:|
| 천장 반사율 | 70 |
| 벽 반사율 | 50 |
| 바닥 반사율 | 10 |

주의: 현재 필요등수 계산식에는 반사율이 직접 들어가지 않는다. 조명률이 이미 공간 조건과 반사 특성을 대표하는 값으로 쓰인다.
